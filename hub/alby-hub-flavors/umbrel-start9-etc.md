# ⚡ Umbrel, Start9, etc

Learn how to install Alby Hub on sovereign computing systems, personal cloud OSs and existing lightning nodes.

Currently Alby Hub is available for Umbrel, Start9, RaspiBlitz, MyNode, CasaOS, Unraid, etc.\
\
Are we missing one option? [Let us know](https://feedback.getalby.com/-alby-hub-request-a-feature)

***

## Umbrel

#### Option 1: Alby Hub self-custodial standalone app using its internal LDK node

**Your Benefits:**

1. **Encrypted remote backups**: recover your bitcoin incl. all channels anytime
2. **Very resource efficient**: Requires minimal storage capacities

Just go to the App Store and install Alby Hub, with no requirements or other app installed previously. It'll work "right out of the box".

{% embed url="https://demos.getalby.com/demo/cmhmn7zwk09bsdqxaxsuc4ape?utm_source=link" %}

Note: If you’ve already installed the Lightning Node app on Umbrel, follow these steps instead:

1. You can either choose Get started (LND) or "Advanced Setup". Click **“Advanced Setup”**
2. Connect your Alby account or choose “Maybe Later”
3. Click **“Create Wallet with Custom Node”**
4. Set a password
5. Select the **“LDK”** option

#### Option 2: Alby Hub using Umbrel's Lightning App (LND) as its funding backend.

Just go to the Umbrel's App Store, then look for the Alby Hub under the "Bitcoin" Section. You need Bitcoin core & Lightning App already installed. Click on Alby Hub & Install.
