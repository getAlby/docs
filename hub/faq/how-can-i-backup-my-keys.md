# How can I backup my keys

{% hint style="info" %}
This guide only refers to the standard Alby Hub version. If you use Alby Hub on top of Umbrel, Start9 and any other external lightning node backend, you'll receive your backup seed phrase from that lightning backend directly.&#x20;
{% endhint %}

Alby Hub contains a bitcoin onchain wallet and a lightning wallet secured by private keys. Only you have access to these keys. We cannot help you to recover it. That's why it is very important to store a backup of your keys: Go to 'Settings', select 'Key Backup' and enter the unlock password to access your recovery phrase:&#x20;

{% hint style="warning" %}
* Your recovery phrase is a set of 12 words that backs up your wallet&#x20;
* Make sure to write them down somewhere safe and private&#x20;
* If you lose access to your recovery phrase, you will lose access to your funds
{% endhint %}
